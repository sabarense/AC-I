02.) Fazer as conversões de binário para decimal:
	
	02a.)
		a.) 10110(2) = X(10)
			
			0x2^0 + 1x2^1 + 1x2^2 + 0x2^3 + 1x2^4
			0     + 2     + 4     + 0     + 16  = 22
			  
		b.) 11011(2) = X(10)
		
			1x2^4 + 1x2^3 + 0x2^2 + 1x2^1 + 1x2^0
			16    + 8   +   0   +   2     + 1 = 27
			
		c.) 100100(2) = X(10)
			
			0x2^0 + 0x2^1 + 1x2^2 + 0x2^3 + 0x2^4 + 1x2^5
			0     + 0     + 4     + 0     + 0     + 32 = 36
			
		d.) 101001(2) = X(10)

			1x2^0 + 0 + 0 + 1x2^3 + 0 + 1x2^5
			1     +         8     + 0 + 32 = 41
			
		e.) 110111(2) = X(10)
		
			1x2^0 + 1x2^1 + 1x2^2 + 0 + 1x2^4 + 1x2^5
			1     + 2     + 4     +     16    + 32 = 55

04.) Fazer as conversões de base entre as bases indicadas por agrupamento:

	04a.)
		a.) 10101(2) = X(4)
			
			01 -> 1
			01 -> 1
			1 -> 1
			
			10101(2) = 111(4)
		
		b.) 011010(2) = X(8)
			
			010 -> 0 + 1x2^1 + 0 = 2 
			011 -> 1x2^0 + 1x2^1 + 0 = 3
			
			011010(2) = 32(8)
		
		c.) 00100111(2) = X(16)
			
			0111 -> 1x2^0 + 1x2^1 + 1x2^2 + 0 = 7
			0010 -> 0 + 1x2^1 + 0 + 0 = 2
			
			00100111(2) = 27(16)
			
			27/16 -> 1 e resta 11 -> B
			1/16 -> 0 e resta 1 -> 1
			
			00100111(2) = 1B(16)
			
		d.) 101001(2) = X(8)
			
			001 -> 1
			101 -> 1x2^0 + 0 + 1x2^2 = 5
			
			011010(2) = 51(8)
			
		e.) 101101(2) = X(4)
		
			01 -> 1
			11 -> 1x2^0 + 1x2^1 + 0 = 3
			10 -> 0 + 1x2^1 + 0 = 2 
			
			101101(2) = 231(4)
